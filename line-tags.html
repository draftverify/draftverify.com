Pillar: Line Tags

Line Tags

Unambiguous identity for NA lines using durable tags with unique IDs and defined placement rules.

Purpose

Instant verification

Tags provide a tamper-evident, visual confirmation at every touchpoint.

Placement

Consistent locations

Apply at keg, trunk/line access, and faucet end per the Standard to ensure visibility during changeovers.

Format

Unique ID + NA mark

Each tag carries a unique identifier and clear NA designation for audit trails and staff checks.

Care

Durability

Moisture-resistant materials; replace if damaged or illegible to maintain audit readiness.

Reference

See the Standard

Requirements and examples are specified in DraftVerify Standard v1.0.
